# Agenda


### Acessar o diretorio e atualizar o composer
```
$cd Agenda/
$composer update
```


### Criando arquivo .env
```
$cp .env.example .env 
```

### Gerando nova key para o ambiente
```
$php artisan key:generate
```

### Executar as migrations para criação do banco
```
$php artisan migrate
```

### Executando projeto
```
$php artisan serve
```
